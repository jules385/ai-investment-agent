📖系统概述

核心价值主张

本系统是一套运行在 Claude Code 上的多智能体投研协作平台。它模拟买方投研团队的完整工作流——从行业判断到买卖决策，由 7 个 AI 角色分工协作完成。

不是"一个 AI 帮你炒股"，而是"一个 AI 团队按照专业投资机构的标准流程为你工作"。

适合谁用

✅ 个人投资者需要系统化的投研流程   ✅ 买方分析师需要快速覆盖多标的   ✅ 想用 AI 辅助而非替代投资决策的人

核心理念：研究驱动决策，纪律约束情绪。四维交叉验证，数据支撑每一个观点。

首席分析师不亲自搜索数据、不调用 MCP、不写分析内容。唯一工作是编排。初次覆盖的目标是决定是否进入白名单，非给出精确买卖点。最终输出四档评级：🟢 核心持仓 🟡 卫星持仓 ⚪ 观察名单 🔴 排除。

Track B：持续跟踪

五级周期体系，不同周期各分析师的参与深度不同：

跟踪周期基本面情绪面筹码面技术面核心问题
周度简要（边际监控）简要标准标准"这周需要行动吗？"
月度简要（边际监控）标准标准标准"中期趋势强化还是弱化？"
季度深度（财务核心）简要标准简要"财报验证还是证伪了逻辑？"
半年深度（逻辑重验）简要标准简要"行业格局有无结构性变化？"
年度深度（全面刷新）简要标准简要"今年的α来自哪里？"

周度（最活跃周期）

基本面仅做边际监控——检查行业重大新闻和个股公告，不重做完整估值。技术面是周度的主角（日线+60分钟双框架），输出本周是否需要行动。四步递进决策：基本面边际检查 → 技术面信号 → 情绪面验证 → 筹码面确认。

月度 / 季度 / 半年&年度

月度：四维全开，核心问题为"中期趋势强化还是弱化？"，输出仓位调整建议。季度：财报后最关键验证节点。基本面回归深度聚焦财务验证，估值更新。半年&年度：全面复盘。深度刷新行业格局和竞争地位，重新验证投资逻辑。

⚡ 临时深度分析触发机制：如果在周度或月度边际监控中发现重大负面信号（实控人大额减持、核心技术被替代、政策转向），立即升级为全深度基本面分析，不等待下一个定期节点。

📊四维分析体系

维度分析内容核心产出物最低标准

🟠 筹码面 股东结构变动（≥3季度对比）+ 主力资金流向 + 北向+融资融券交叉分析 + 龙虎榜完整记录（≥4条）+ 筹码分布 + 五维综合评分 资金合力方向 + 对手盘压力 + 建仓/出货阶段 ≥2000字 / ≥6 MCP工具
🟢 技术面 5周期趋势定位（月/周/日/60m/30m）+ 13项指标速览 + 7信号类型买卖评估 + 8锚点价格体系 + 3情景推演 买入区间 + 止损位 + 目标价 + 风险收益比 ≥2000字 / ≥8 MCP工具
🟣 情绪面 市场情绪（6项子指标）+ 资金面（主力/散户对比）+ 个股舆情（≥10条）+ 散户行为逆向解读（4类型抽样）+ 加权温度计 情绪温度计（0-100）+ 陷阱识别 + 左侧/右侧策略建议 ≥2000字 / ≥3 MCP工具

每个维度输出标准化 YAML 信号块，四维信号可直接合并加权。机械化的信号格式确保不因个人偏好扭曲决策。

🚀快速开始

环境要求

Python 3.10+  |  Claude Code（桌面版/CLI/IDE 插件） |  DeepSeek API Key 或 Anthropic API Key

三步安装

# 1. 克隆仓库
git clone https://github.com/jules385/ai-investment-agent.git
cd ai-investment-agent

# 2. 安装 Python 依赖
pip install -r requirements.txt

# 3. 运行安装脚本（自动配置 Skills / Commands / MCP）
python install.py

配置

① 编辑 ~/.claude/settings.json，填入你的 API Key（参考 workspace-template/.claude/settings.json.template）

② 复制 workspace-template/ 到你的工作目录（如 ~/ai-investment/）

③ 在 reports/白名单.md 中添加你想覆盖的标的

⭐核心优势

1. 多维度交叉验证

四位分析师从完全不同的数据源和分析框架独立得出结论。基本面看"值不值"，情绪面看"怕不怕"，筹码面看"谁在买"，技术面看"何时买"。四维交叉，避免单一维度偏见。

2. 数据驱动决策

每个观点必须有数据支撑，每个趋势必须有3-5年历史数据验证。40+量化指标覆盖供给端/需求端/竞争格局。数据分析师对每份报告进行逐条数据质量检查，缺口透明呈现。

3. 标准化流程，可回溯可复盘

每次分析遵循固定工作流，每个标的独立存档。初次覆盖、周度/月度/季度/半年/年度跟踪各有标准模板。决策日志记录每一次操作的理由，事后可复盘。

4. 零成本数据基础设施

全部基于 AKShare 免费开源数据，无需 Bloomberg/Wind 等昂贵终端。自建 MCP Server 封装关键接口，内置请求节流和自动回退。1000+ API 覆盖全品类金融数据。

5. 模块化可扩展

7 个分析师 Skill + 2 个 MCP Server 均为独立模块。可随时新增数据源（Wind/同花顺）、新增分析维度（ESG/量化因子）、替换底层 LLM（Claude/GPT/DeepSeek）。

6. 知识沉淀与复用

每个标的的研究数据以标准化格式存档（CSV序列+Markdown报告），形成可检索的研究知识库。新覆盖标的时可直接复用行业模板和可比公司数据，研究效率持续提升。
