2

3

4

估值判断

PE分位数 + 可比估值 + DCF区间

5

白名单决策

六维加权打分 → 🟢核心/🟡卫星/⚪观察/🔴排除

📡 标准化信号接口

四位分析师通过统一的 YAML 信号格式与总分析师通信，实现解耦和高效合成

# 每个分析师报告中的标准化信号块 —— 总分析师直接消费
signal:
  direction: bullish          # bullish | neutral | bearish
  strength:  moderate         # strong | moderate | weak
  confidence: 78             # 0-100 分析师对自身判断的把握
  score:      77             # 0-100 标准化评分

key_evidence:                 # 3条核心证据
  - "护城河极深：唯一总体资质..."
  - "财务反转：净利润V型反弹"
  - "估值合理：PE 22x"

red_flags:                    # 危险信号（如有则总分析师一票否决）
  - "实控人减持套现10亿"

catalyst:                     # 未来催化剂
  - "十五五规划落地"

✨ 核心优势

与传统投研方式相比，这套 AI Agent 系统在以下维度具有显著优势

🔄 多维度交叉验证

四位分析师从完全不同的数据源和分析框架独立得出结论。基本面看"值不值"，情绪面看"怕不怕"，筹码面看"谁在买"，技术面看"何时买"。四维交叉，避免单一维度偏见导致的决策失误。

⚡ 并行效率

四位分析师通过 Claude Code Subagent 并行工作，一次"月度跟踪"只需几分钟即可完成，而人工完成同等深度研究需要 4-6 小时。特别适合覆盖多标的的买方分析师。

📐 标准化决策流程

每次分析遵循固定的工作流——从数据获取到信号输出，从矛盾检测到加权决策。消除情绪化决策，确保每一笔投资都经过相同的严谨流程。决策日志可回溯、可复盘。

🆓 零成本数据源

全部基于 AKShare 免费开源数据，无需 Bloomberg/Wind 等昂贵终端。1000+ API 覆盖行情、财务、资金流向、龙虎榜、筹码分布、融资融券等，数据广度超过多数付费平台。

🧬 模块化可扩展

63 个金融 Skill + 5 个分析师 Skill + 2 个 MCP Server 均为独立模块。可随时新增数据源（Wind/同花顺）、新增分析维度（ESG/量化因子）、替换底层 LLM（DeepSeek/Qwen/Claude）。

📚 知识沉淀与复用

每个标的的研究数据以标准化格式存档（CSV序列+Markdown报告），形成可检索的研究知识库。新覆盖标的时可直接复用行业模板和可比公司数据，研究效率持续提升。
