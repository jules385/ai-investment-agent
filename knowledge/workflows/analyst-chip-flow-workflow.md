# 筹码面分析师（Chip & Flow Analyst）工作流

> **角色定位**：买方资金分析专家 —— 从大资金的筹码博弈角度，判断"谁在买、谁在卖、谁在锁仓、谁在出货"。
> **核心能力**：股东结构 → 主力资金流 → 北向/融资融券 → 龙虎榜 → 筹码成本分布
> **关键产出**：资金合力方向 + 对手盘压力评级 + 建仓/出货阶段判断

---

## 一、工作模式总览

筹码面分析师有**两种工作模式**：

```
┌─────────────────────────────────────────────────────────┐
│                    筹码面分析师双模式                     │
├─────────────────────────┬───────────────────────────────┤
│  模式A：深度分析模式     │  模式B：快速扫描模式            │
│  适用：初次覆盖/月度/    │  适用：周度                     │
│        季度/半年/年度    │                               │
│  耗时：30-45分钟/标的    │  耗时：5-10分钟/标的            │
│  目标：全面筹码结构分析  │  目标：发现资金异动             │
└─────────────────────────┴─────────────────────────────┘
```

### 1.1 模式触发条件

| 跟踪周期 | 模式 | 深度等级 | 说明 |
|---------|------|---------|------|
| 初次覆盖 | 🅰️ 深度 | ★★★★☆ | 全面筹码结构画像 |
| 周度 | 🅱️ 快速 | ★★☆☆☆ | 资金异动扫描 |
| 月度 | 🅰️ 深度 | ★★★★☆ | 主力月度行为分析 |
| 季度 | 🅰️ 深度 | ★★★★☆ | 配合财报的筹码变化 |
| 半年 | 🅰️ 深度 | ★★★★☆ | 长期资金趋势 |
| 年度 | 🅰️ 深度 | ★★★★☆ | 年度筹码结构变迁 |

## 二、模式 A：深度分析流程（五步法）

```
Step 1           Step 2           Step 3          Step 4          Step 5
股东结构分析 → 主力资金流向 → 北向+融资融券 → 龙虎榜分析 → 筹码分布+合力判断
  (10min)         (10min)          (10min)         (10min)         (10min)
```

### Step 1：股东结构分析（10分钟）

#### 1.1 数据获取

| 数据需求 | 来源 | 调用 |
|---------|------|------|
| 前十大股东 | AKShare | `stock_main_stock_holder(symbol)` |
| 流通股东 | AKShare | `stock_circulate_stock_holder(symbol)` |
| 机构持仓 | AKShare | `stock_fund_stock_holder(symbol)` |
| 股东人数变化 | AKShare | `stock_shareholder_change_ths(symbol)` |

#### 1.2 分析框架

```
股东结构画像：

1. 股权集中度
   ├── 第一大股东：[名称]，持股 [X%]
   ├── 前三大股东合计：[X%]
   ├── 前十大股东合计：[X%]
   └── 判断：
       前十大 > 60% → 高度集中（筹码锁定好，但流动性差）
       前十大 40-60% → 适度集中（健康）
       前十大 < 40% → 分散（无主力控盘）

2. 股东类型构成
   | 股东类型 | 占比 | 代表 | 特点 |
   |---------|------|------|------|
   | 实控人/一致行动人 | X% | [名称] | 核心锁定筹码 |
   | 机构投资者(公募/私募) | X% | [基金名] | 专业长线资金 |
   | 外资(QFII/北向) | X% | [名称] | 价值发现者 |
   | 国家队(汇金/证金/养老金) | X% | [名称] | 稳定器 |
   | 自然人/其他 | X% | — | 散户筹码 |

3. 股东户数趋势（过去4个季度）
   | 报告期 | 股东户数 | 环比变化 | 户均持股(万) | 信号 |
   |--------|---------|---------|-------------|------|
   | 最新 | X | — | X | |
   | Q-1 | X | +/-X% | X | |
   | Q-2 | X | +/-X% | X | |
   | Q-3 | X | +/-X% | X | |
   
   判断：
   股东户数 ↓ + 户均持股 ↑ → ✅ 筹码集中（主力在吸筹）
   股东户数 ↑ + 户均持股 ↓ → ⚠️ 筹码分散（主力在出货）

4. 实控人行为 ⚠️（最重要）
   ├── 近1年增减持：增持 X / 减持 X
   ├── 是否有质押：[是/否]，质押比例 [X%]
   └── 判断：
       实控人大额减持(>1%总股本) → 🔴 重大危险信号
       实控人增持 → ✅ 管理层信心
       高质押(>50%) → ⚠️ 潜在爆仓风险
```

### Step 2：主力资金流向分析（10分钟）

#### 2.1 数据获取

| 数据需求 | 来源 | 调用 |
|---------|------|------|
| 个股资金流向 | AKShare | `stock_individual_fund_flow(symbol)` |
| 主力净流入 | AKShare | `stock_main_fund_flow(symbol)` |
| 大单成交 | AKShare | `stock_fund_flow_big_deal(symbol)` |

#### 2.2 分析框架

```
主力资金流向分析：

1. 近期资金流向（日度，近20个交易日）
   | 日期 | 主力净流入(亿) | 超大单净流入 | 大单净流入 | 中单 | 小单 |
   |------|-------------|------------|----------|------|------|
   | D-1 | +/-X | +/-X | +/-X | +/-X | +/-X |
   | D-2 | ... | | | | |
   | ... | | | | | |

2. 主力行为模式判断

   计算近20日主力累计净流入：
   ├── 累计净流入 > 流通市值 0.5% → 🟢 主力积极吸筹
   ├── 累计净流入 0~0.5% → 🟡 主力温和参与
   ├── 累计净流出 0~-0.5% → 🟡 主力温和减仓
   └── 累计净流出 > 流通市值 0.5% → 🔴 主力积极出货

3. 大单与小单的博弈

   | 模式 | 主力(超大+大单) | 散户(中小单) | 含义 |
   |------|--------------|------------|------|
   | 主力吸筹 | 净买入 ↑ | 净卖出 ↑ | ✅ 散户恐慌卖出，主力接盘 |
   | 主力出货 | 净卖出 ↑ | 净买入 ↑ | 🔴 主力拉高出货给散户 |
   | 合力上涨 | 净买入 ↑ | 净买入 ↑ | ✅ 大小资金共振做多 |
   | 合力下跌 | 净卖出 ↑ | 净卖出 ↑ | ⚠️ 恐慌性出逃 |

4. 近5日 vs 近20日对比
   ├── 5日净流入 > 20日均值 → 近期加速流入
   └── 5日净流出 > 20日均值 → 近期加速流出
```

### Step 3：北向资金 + 融资融券（10分钟）

#### 3.1 数据获取

| 数据需求 | 来源 | 调用 |
|---------|------|------|
| 北向资金持仓 | AKShare | `stock_hsgt_individual_em(symbol)` |
| 北向资金历史流向 | AKShare | `stock_hsgt_hist_em(symbol)` |
| 融资融券余额 | AKShare | `stock_margin_detail_sse(symbol)` 或 `stock_margin_detail_szse(symbol)` |

#### 3.2 分析框架

```
北向资金分析：

1. 北向持仓趋势
   ├── 当前持仓市值：[X亿]，占自由流通市值 [X%]
   ├── 近1月净买入：[+X亿/-X亿]
   ├── 近3月净买入：[+X亿/-X亿]
   └── 近1年净买入：[+X亿/-X亿]

   判断：
   北向持续流入 → 🟢 外资长期看好，价值发现
   北向持续流出 → ⚠️ 外资不认可当前估值或逻辑
   北向大幅波动 → ⚪ 短期交易型外资进出

2. 北向持仓占自由流通市值
   ├── >5% → 外资重仓，重要边际买家
   ├── 2-5% → 外资适度参与
   └── <2% → 外资关注度低

融资融券分析：

3. 融资余额趋势
   ├── 当前融资余额：[X亿]
   ├── 近1月变化：[+X%/-X%]
   └── 融资余额占流通市值：[X%]

   判断：
   融资余额快速上升(月+20%以上) → ⚠️ 杠杆资金涌入，助涨助跌
   融资余额持续下降 → 杠杆资金离场，风险偏好降低
   融资占比 > 流通市值5% → 🔴 杠杆风险较高，踩踏风险

4. 北向 vs 融资对比（资金性质的较量）
   | 北向 | 融资 | 合力判断 |
   |------|------|---------|
   | 流入 | 流入 | 🟢 长线+杠杆共振做多 |
   | 流入 | 流出 | 🟡 长线看好但短线谨慎 |
   | 流出 | 流入 | ⚠️ 外资撤退+散户杠杆接盘（危险） |
   | 流出 | 流出 | 🔴 资金全面撤退 |
```

### Step 4：龙虎榜分析（10分钟）

#### 4.1 数据获取

| 数据需求 | 来源 | 调用 |
|---------|------|------|
| 龙虎榜明细 | AKShare | `stock_lhb_stock_detail_em(symbol)` |
| 龙虎榜机构统计 | AKShare | `stock_lhb_jgstatistic_em(symbol)` |
| 龙虎榜营业部统计 | AKShare | `stock_lhb_yybph_em(symbol)` |

#### 4.2 分析框架

```
龙虎榜分析（近3个月）：

1. 上榜频率
   近3个月上榜 [X] 次
   ├── ≥5次 → 高频上榜，游资活跃
   ├── 2-4次 → 偶尔上榜
   └── <2次 → 机构锁仓型，短线关注度低

2. 最近一次龙虎榜席位分析
   | 席位 | 类型 | 买入(亿) | 卖出(亿) | 净额(亿) |
   |------|------|---------|---------|---------|
   | 买一 | 机构/游资 | X | — | +X |
   | 买二 | 机构/游资 | X | — | +X |
   | ... | | | | |
   | 卖一 | 机构/游资 | — | X | -X |
   | ... | | | | |

   买卖力量对比：
   买入前五合计：[X亿] vs 卖出前五合计：[X亿] → 净买入/卖出 [X亿]

3. 机构 vs 游资行为
   ├── 机构席位累计净买入 [X亿] → 机构在买/卖
   ├── 游资席位累计净买入 [X亿] → 游资方向
   └── 判断：
       机构买 + 游资买 → ✅ 一致做多
       机构买 + 游资卖 → ⚪ 短线资金换手给长线
       机构卖 + 游资买 → ⚠️ 机构出货给游资（危险）
       机构卖 + 游资卖 → 🔴 一致出逃

4. 龙虎榜买卖比
   买入总额/卖出总额 = [X]
   ├── >1.5 → 买方主导，强势
   ├── 1.0-1.5 → 偏多
   ├── 0.7-1.0 → 偏空
   └── <0.7 → 卖方主导，弱势
```

### Step 5：筹码成本分布 + 资金合力判断（10分钟）

#### 5.1 数据获取

| 数据需求 | 来源 | 调用 |
|---------|------|------|
| 筹码分布 | AKShare | `stock_cyq_em(symbol)` |
| 日均换手率 | AKShare | K线数据计算 |

#### 5.2 分析框架

```
筹码成本分布分析：

1. 筹码分布形态
   使用 stock_cyq_em 获取：
   ├── 获利盘比例：[X%]
   ├── 套牢盘比例：[X%]
   ├── 筹码密集区1：[X元-Y元]（集中度最高）
   ├── 筹码密集区2：[X元-Y元]
   └── 平均持仓成本：[X元]

2. 筹码形态判断
   | 形态 | 特征 | 含义 |
   |------|------|------|
   | 单峰密集 | 筹码集中于窄区间 | 变盘前兆，方向待选择 |
   | 双峰 | 上下两个密集区 | 上方套牢盘 vs 下方获利盘博弈 |
   | 多峰分散 | 筹码分布散乱 | 无明确主力，随大盘波动 |
   | 低位单峰 | 密集区在低位 | 主力吸筹完毕，等待拉升 |
   | 高位单峰 | 密集区在高位 | ⚠️ 主力已派发，筹码转移到散户 |

3. 筹码支撑与阻力
   基于筹码分布确定关键价位：
   ├── 筹码支撑位：[X元]（密集区下沿/平均成本）
   ├── 筹码阻力位：[X元]（密集区上沿/套牢盘密集区）
   └── 如果价格在成本线以下 → 大部分持仓者亏损，惜售心理强

4. 换手率趋势
   近20日日均换手率：[X%]
   ├── 换手率 < 1% → 筹码高度锁定，主力控盘
   ├── 换手率 1-3% → 正常换手
   ├── 换手率 3-8% → 交易活跃
   └── 换手率 > 8% → ⚠️ 异常放量，可能变盘
```

#### 5.3 综合资金合力判断

```
五维综合判断：

┌────────────────┬──────────┬──────────┬──────────┐
│ 维度            │ 信号      │ 方向      │ 置信度    │
├────────────────┼──────────┼──────────┼──────────┤
│ 1.股东结构      │ 集中/分散 │ bullish/ │ 高/中/低  │
│                │          │ bearish  │          │
│ 2.主力资金流向  │ 流入/流出 │ bullish/ │ 高/中/低  │
│                │          │ bearish  │          │
│ 3.北向+融资     │ 共振/背离 │ bullish/ │ 高/中/低  │
│                │          │ bearish  │          │
│ 4.龙虎榜        │ 买方/卖方│ bullish/ │ 高/中/低  │
│                │ 主导     │ bearish  │          │
│ 5.筹码分布      │ 低位/高位│ bullish/ │ 高/中/低  │
│                │ 密集     │ bearish  │          │
├────────────────┼──────────┼──────────┼──────────┤
│ 综合合力判断    │          │          │          │
└────────────────┴──────────┴──────────┴──────────┘

合力方向：
  🟢 做多合力：≥4个维度看多，无明显矛盾
  🟡 偏多：3个维度看多，存在1-2个矛盾
  ⚪ 中性平衡：多空维度各半
  🟠 偏空：3个维度看空
  🔴 出货信号：≥4个维度看空，主力明确出货

建仓/出货阶段判断：
  ├── 吸筹阶段：股东户数↓ + 主力净流入 + 低位筹码密集
  ├── 锁仓拉升：筹码锁定(低换手) + 主力净流入温和 + 北向跟随
  ├── 高位派发：股东户数↑ + 主力净流出 + 高位单峰 + 龙虎榜机构卖
  └── 出货完成：筹码分散 + 散户主导 + 北向撤离

对手盘压力评级：
  │对手盘压力│ 条件                             │ 操作含义            │
  ├─────────┼──────────────────────────────────┼────────────────────┤
  │ 🔴 极重   │ 主力大规模出货+高位套牢盘密集       │ 避免买入，考虑止损  │
  │ 🟠 较重   │ 上方有密集套牢区+北向流出           │ 谨慎，小仓位试探    │
  │ 🟡 适中   │ 存在分歧但无压倒性空方              │ 可正常操作         │
  │ 🟢 较轻   │ 筹码集中+主力锁仓+无套牢压力        │ 可积极操作         │
  └─────────┴──────────────────────────────────┴────────────────────┘
```

## 三、模式 B：快速扫描流程（周度）

```
周度筹码快速扫描（5-10分钟）：

1. 本周主力资金净流向（stock_individual_fund_flow，近5日）
   Q: 本周主力在买还是卖？规模多大？
   └→ 🟢/🟡/🔴

2. 北向资金周流向（stock_hsgt_hist_em）
   Q: 北向本周净买还是净卖？
   └→ 🟢/🟡/🔴

3. 融资余额周变化
   Q: 杠杆资金在加还是减？
   └→ 🟢/🟡/🔴

4. 本周是否上龙虎榜？
   Q: 如有，买卖力量如何？
   └→ 🟢/🟡/🔴

5. 有无大宗交易/增减持公告？
   Q: 大股东/机构是否在大宗交易？
   └→ 🟢/🟡/🔴

综合：本周资金面有无重大异动？
  ├── 有异动 → 标注信号方向，传递给总分析师
  └── 无异动 → 维持上次深度分析结论
```
