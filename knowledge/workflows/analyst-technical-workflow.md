# 技术面分析师（Technical Analyst）工作流

> **角色定位**：买方技术分析专家 —— 通过量价关系和指标信号，判断"什么时候买、什么价格买、什么时候卖"。
> **核心能力**：趋势判断 → 多指标共振 → 买卖信号 → 止损止盈定价
> **专属工具**：tech-analysis MCP Server（10大指标工具）

---

## 一、专属 MCP 工具速查

技术面分析师拥有专用 MCP Server，所有指标计算通过工具调用完成：

```
tech-analysis MCP 工具清单：

趋势类：
  compute_ma(symbol, period, days, ma_windows)    → 均线排列 + 价格偏离度
  compute_macd(symbol, period, days)               → MACD金叉死叉 + 背离

摆动类：
  compute_rsi(symbol, period, days, rsi_period)    → RSI超买超卖 + 趋势
  compute_kdj(symbol, period, days, k_period)      → KDJ金叉死叉 + 超买超卖

波动类：
  compute_bollinger(symbol, period, days)           → 布林带位置 + 收窄/扩张
  compute_atr(symbol, period, days)                 → 波动率 + 止损参考

量价类：
  compute_volume(symbol, period, days)              → 量比 + 量价关系

形态类：
  detect_patterns(symbol, period, days)              → K线形态识别
  support_resistance(symbol, period, days)           → 支撑/阻力位

综合类：
  technical_score(symbol, period, days)              → 综合评分(0-100) + 评级
```

## 三、分析流程

### 标准分析流程（四步法）

```
Step 1              Step 2              Step 3            Step 4
多周期趋势定位  →  多指标信号扫描  →  买卖信号生成  →  价格锚点设定
  (5min)             (10min)            (10min)           (5min)
```

### Step 1：多周期趋势定位（5分钟）

> **原则**：大周期定方向，小周期找买点。如果大小周期方向矛盾，以大周期为准。

#### 1.1 多时间框架调用

```
第一步：获取周线数据（定中期方向）
  compute_ma(symbol, "weekly", 100)      → 周线均线排列
  compute_macd(symbol, "weekly", 100)    → 周线MACD位置

第二步：获取日线数据（定短期信号）
  compute_ma(symbol, "daily", 120)       → 日线均线排列
  compute_macd(symbol, "daily", 120)     → 日线MACD
  technical_score(symbol, "daily", 250)  → 日线综合评分
```

#### 1.2 趋势定位矩阵

```
多周期趋势定位：

┌──────────┬──────────┬──────────┬──────────────────────┐
│ 周线趋势  │ 日线趋势  │ 趋势定位  │ 策略含义              │
├──────────┼──────────┼──────────┼──────────────────────┤
│ 多头排列  │ 多头排列  │ 🟢 主升   │ 持有为主，回调加仓     │
│ 多头排列  │ 空头排列  │ 🟡 回调   │ 等待日线企稳再介入     │
│ 空头排列  │ 多头排列  │ 🟡 反弹   │ 短期参与，设紧止损     │
│ 空头排列  │ 空头排列  │ 🔴 主跌   │ 回避或轻仓做空         │
│ 交织      │ 多头排列  │ 🟡 转多   │ 关注突破确认           │
│ 交织      │ 空头排列  │ 🟡 转空   │ 警惕破位风险           │
└──────────┴──────────┴──────────┴──────────────────────┘

特殊情形：
  周线多头 + 日线底背离 → ⭐ 最佳中期买点
  周线空头 + 日线顶背离 → ⚠️ 最后逃命机会
```

### Step 2：多指标信号扫描（10分钟）

调用所有技术指标工具，汇总信号：

#### 2.1 指标调用

```python
# 一次性调用所有日线指标
compute_macd(symbol, "daily", 250)        # MACD
compute_rsi(symbol, "daily", 250, 14)     # RSI
compute_kdj(symbol, "daily", 250, 9)      # KDJ
compute_bollinger(symbol, "daily", 250)   # 布林带
compute_atr(symbol, "daily", 120, 14)     # ATR波动率
compute_volume(symbol, "daily", 120)      # 量价关系
detect_patterns(symbol, "daily", 250)     # K线形态
support_resistance(symbol, "daily", 250)  # 支撑阻力
technical_score(symbol, "daily", 250)     # 综合评分
```

#### 2.2 信号汇总表

```
多指标信号汇总：

| 指标类别 | 指标 | 数值 | 信号 | 多/空 | 权重 |
|---------|------|------|------|------|------|
| 趋势 | 均线排列 | 多头/交织/空头 | XX | ±1 | 20% |
| 趋势 | MACD | DIF/DEA/HIST | 金叉/死叉/持有 | ±1 | 20% |
| 摆动 | RSI(14) | X | 超买/中性/超卖 | ±1 | 15% |
| 摆动 | KDJ | K/D/J | 金叉/死叉 | ±1 | 10% |
| 波动 | 布林带 | 位置X% | 轨上/中/下 | ±1 | 10% |
| 量价 | 成交量 | 量比X | 放量/缩量 | ±1 | 10% |
| 形态 | K线形态 | [形态名] | 看涨/看跌 | ±1 | 10% |
| 波动 | ATR | X | 高/中/低波动 | ±1 | 5%  |

信号共振强度 = Σ(信号方向 × 权重)
  ├── ≥ 0.6 → 强看多信号共振
  ├── 0.3~0.6 → 弱看多
  ├── -0.3~0.3 → 信号矛盾，无明确方向
  ├── -0.6~-0.3 → 弱看空
  └── ≤ -0.6 → 强看空信号共振
```

### Step 3：买卖信号生成（10分钟）

#### 3.1 买入信号强度分级

```
买入信号评估：

🟢🟢 强买入（高确信度）— 需 ≥3 个条件同时满足：
  □ 周线+日线趋势一致看多（均线多头或MACD零轴上）
  □ MACD金叉（或零轴上方二次金叉）
  □ RSI在40-60区域拐头向上（非超买区）
  □ 成交量放大（量比>1.2）
  □ K线形态出现底部反转信号（锤子线/启明星/看涨吞没）
  □ 价格在布林带中轨附近或刚从下轨反弹

🟢 弱买入（中低确信度）— 需 ≥1 个条件：
  □ 日线看多但周线中性
  □ RSI超卖（<30）拐头
  □ 单指标金叉（仅MACD或仅KDJ）
  □ 价格触及布林下轨
  □ 缩量下跌（抛压减轻）

⚪ 无信号 — 维持观望：
  □ 指标多空矛盾
  □ 均线交织无方向
  □ 成交量异常萎缩

🔴 卖出信号：
  □ MACD死叉（尤其是零轴上方死叉）
  □ RSI超买（>70）拐头向下
  □ KDJ高位死叉
  □ 价格跌破MA60
  □ 放量下跌（主力出逃）
  □ K线顶部反转形态（吊颈线/黄昏星/看跌吞没）
```

#### 3.2 信号强度与仓位对应

| 信号强度 | 操作 | 仓位变化 | 止损宽松度 |
|---------|------|---------|----------|
| 🟢🟢 强买入 | 加仓 | +20~30% | 正常(2倍ATR) |
| 🟢 弱买入 | 试探性加仓 | +5~10% | 紧凑(1.5倍ATR) |
| ⚪ 无信号 | 维持 | 0% | — |
| 🔴 卖出信号 | 减仓/清仓 | -20~50% | — |

---

### Step 4：价格锚点设定（5分钟）

> 技术分析的核心价值不在于预测方向，而在于**提供具体的操作价格**。

#### 4.1 多维度支撑/阻力综合

```
价格锚点体系：

1. 买入价格区间（三层）
   激进买点（当前可买）：当前价 ± 1%
   合理买点（回调买入）：最近的支撑位
   保守买点（最佳买点）：强支撑位（MA60 / 布林下轨 / 前低共振）

2. 止损位设定（取最严格的一个）
   方法A：ATR止损 = 买入价 - 2×ATR(14)
   方法B：均线止损 = MA60（趋势投资者最后防线）
   方法C：结构止损 = 最近的前低（跌破结构转空）
   → 取 A/B/C 中最靠近买入价的（亏损最小）

3. 止盈位设定（分段）
   第一止盈：最近阻力位（到达减仓1/3）
   第二止盈：前高 / 布林上轨（到达再减1/3）
   移动止盈：随价格上涨，止损上移至成本价以上
```

#### 4.2 风险收益比计算

```
风险收益比 = (目标价 - 买入价) / (买入价 - 止损价)

  ≥ 3:1 → ⭐ 优秀交易机会
  2:1 ~ 3:1 → ✅ 可接受
  1:1 ~ 2:1 → ⚠️ 勉强，需其他维度强力支撑
  < 1:1 → ❌ 不值得参与
```

## 四、各周期适配

### 周度（最活跃周期）

```
周度技术分析标准流程：
  1. 日线 + 60分钟线双时间框架
  2. 全指标扫描（重点关注MACD/RSI/量价变化）
  3. 输出周度信号 + 操作建议
  4. 更新技术评分
```

### 月度

```
月度技术分析：
  1. 周线 + 日线双时间框架
  2. 重点关注均线系统变化（MA20/60/120排列）
  3. 技术评分月度趋势对比
  4. 中期支撑/阻力位更新
```

### 初次覆盖 / 季度 / 半年 / 年度

```
大周期技术分析：
  1. 月线/周线定大趋势位置（历史高/低位区域判断）
  2. 简化指标扫描（只需均线+MACD+RSI）
  3. 重点输出：当前在历史趋势中的位置（底部/中部/顶部）
  4. 给出理想的技术买点区间（不需要精确到日）
```

## 六、核心原则

1. **大周期定方向**：周线 > 日线，日线信号与周线矛盾时，以周线为准
2. **共振提高确信度**：单一指标信号不可靠，≥3个指标同向时才行动
3. **价格锚点是核心价值**：技术分析的价值在于给出具体的"买在哪、损在哪、赚到哪"
4. **止损永远必设**：入场的同时就必须设好止损，没有例外
5. **不预测，只应对**：不对"明天涨还是跌"做判断，只说"如果涨到X怎么做，如果跌破Y怎么做"
