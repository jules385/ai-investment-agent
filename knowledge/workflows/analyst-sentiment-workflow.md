# 情绪面分析师（Sentiment Analyst）工作流

> **角色定位**：买方情绪监测专家 —— 用数据量化市场贪婪与恐慌，回答"现在是该贪婪还是该恐惧"。
> **核心能力**：市场情绪指标 → 个股舆情分析 → 散户行为监测 → 情绪温度计 → 左侧/右侧策略建议
> **独特价值**：其他三位分析师告诉你"买什么"和"什么价买"，情绪面告诉你要不要"逆着人群走"。

## 二、核心概念：情绪温度计

情绪面分析师的核心产出是一个 **0-100 温度计**：

```
情绪温度计量表：

   0                 25                 50                 75                100
   ├─────────────────┼─────────────────┼─────────────────┼─────────────────┤
   🥶               🟢                ⚪                🟡                🔥
 极度恐慌          偏冷              中性              偏热             极度亢奋
   │                 │                 │                 │                 │
   └─ 最优买点 ──────┘                 │                 └─ 减仓信号 ──────┘
      左侧分批布局                       │                 逐步获利了结
                                   维持观望

温度走势比温度绝对值更重要：

  🥶→🟢 从冰点回暖 → ⭐ 最佳右侧买点（情绪刚刚开始修复）
  🟢→🥶 从温和转冷 → 可能只是暂时回调，不必恐慌
  ⚪→🔥 突然升温    → ⚠️ 警惕情绪过热，准备减仓
  🔥→🟡 从亢奋降温 → 🟢 健康的回调，关注再入场机会
  🥶🥶 持续冰点     → 可能基本面出了问题，需交叉验证
  🔥🔥 持续亢奋     → 博傻阶段，远离或设紧止损
```

## 三、模式 A：深度情绪评估（四维法）

```
维度1           维度2           维度3           维度4
市场整体情绪 → 个股舆情分析 → 散户行为监测 → 综合温度计
  (5min)         (10min)         (10min)        (5min)
```

### 维度 1：市场整体情绪（5分钟）

#### 1.1 数据获取

| 指标 | 来源 | 调用 |
|------|------|------|
| 涨停/跌停家数 | AKShare | `stock_zt_pool_em`（涨停池）|
| 上涨/下跌家数比 | AKShare | 通过指数数据分析 |
| 全市场成交额 | AKShare | 指数成交数据 |
| 融资融券余额 | AKShare | `stock_margin_sse` / `stock_margin_szse` |
| 北向资金净流向 | AKShare | `stock_hsgt_fund_flow_summary_em` |
| 央视/经济新闻 | AKShare | `news_cctv` |

#### 1.2 评分体系

```
市场整体情绪打分（满分25分）：

1. 涨跌停对比（5分）
   ├── 涨停≥100 + 跌停≤10 → 5分（亢奋）
   ├── 涨停50-100 → 3分（正常偏多）
   ├── 涨停跌停均衡 → 2分（中性）
   ├── 跌停明显多于涨停 → 1分（偏恐慌）
   └── 跌停≥100 + 涨停≤10 → 0分（极度恐慌）

2. 全市场成交额趋势（5分）
   ├── 成交额持续放大（>20日均量1.5倍）→ 5分（资金涌入）
   ├── 成交额正常（0.8-1.5倍）→ 3分
   └── 成交额萎缩（<0.8倍）→ 1分（无人参与）

3. 融资余额变化趋势（5分）
   ├── 融资快速上升（月+15%以上）→ 5分（杠杆涌入）
   ├── 融资温和变化 → 3分
   └── 融资快速下降（月-10%以上）→ 1分（杠杆撤离）

4. 北向资金情绪（5分）
   ├── 北向连续5日净流入 → 5分
   ├── 北向流入流出交替 → 3分
   └── 北向连续5日净流出 → 1分

5. 新闻/政策情绪（5分）
   ├── 政策利好频出+媒体乐观 → 5分
   ├── 消息面平静 → 3分
   └── 利空消息主导 → 1分

市场情绪得分 = 五项得分之和（满分25，折算为0-100）
```

### 维度 2：个股舆情分析（10分钟）

#### 2.1 数据获取

| 数据需求 | 来源 | 方法 |
|---------|------|------|
| 个股新闻 | AKShare `stock_news_em` | 获取近1周/1月新闻列表 |
| 研报评级 | WebSearch | `"[股票名] 研报 评级 目标价"` |
| 最新公告解读 | WebSearch | `"[股票名] 公告 解读"` |
| 机构调研 | WebSearch | `"[股票名] 机构调研 投资者关系"` |

#### 2.2 分析框架

```
个股舆情分析：

1. 新闻情绪归类
   将获取到的新闻逐条标记情绪倾向：

   | 日期 | 标题 | 情绪 | 影响等级 |
   |------|------|------|---------|
   | D-1 | [标题] | 正面/中性/负面 | 重大/一般/轻微 |

   统计：
   ├── 正面新闻：[X]条，占比 [X%]
   ├── 中性新闻：[X]条，占比 [X%]
   └── 负面新闻：[X]条，占比 [X%]

2. 关键舆情信号识别

   🟢 正面信号：
   □ 多家券商上调评级/目标价
   □ 业绩超预期/预告向好
   □ 重大合同/订单公告
   □ 机构密集调研
   □ 公司回购/管理层增持
   □ 行业政策利好

   🔴 负面信号：
   □ 券商下调评级/目标价
   □ 业绩低于预期/预告向下
   □ 监管问询/处罚
   □ 大股东减持
   □ 负面新闻持续发酵
   □ 行业政策收紧

3. 舆情综合判断
   舆情得分 = 正面新闻占比 × 100（满分100）
   再根据"影响等级"做加权调整：
   ├── 有"重大正面"新闻 → +10
   ├── 有"重大负面"新闻 → -15
   └── 重大负面超过2条 → -25
```

### 维度 3：散户行为监测（10分钟）

#### 3.1 数据获取

| 数据需求 | 来源 | 方法 |
|---------|------|------|
| 散户讨论热度 | WebSearch | `"[股票名] 股吧 讨论"` |
| 雪球社区情绪 | WebSearch | `"[股票名] 雪球 热门"` |
| 龙虎榜散户买卖 | AKShare | `stock_lhb_detail_em`（散户席位） |
| 小单资金流向 | AKShare | `stock_individual_fund_flow`（小单净买卖） |

#### 3.2 分析框架

```
散户行为分析：

1. 讨论热度
   WebSearch 搜索股吧/雪球：
   ├── 讨论量近期大幅上升 → 🟡 关注度提升（可能在顶部或底部）
   ├── 讨论量正常 → ⚪ 关注度稳定
   └── 讨论量稀少 → 🟢 无人关注（可能是底部特征）

2. 散户情绪定性
   从讨论内容判断散户情绪倾向：
   ├── "坚定持有""长期看好" → 🟡 乐观（需要警惕一致性预期）
   ├── "受不了了""割肉走了" → 🟢 恐慌（可能是底部信号）
   ├── 骂声一片/怨气冲天 → 🟢🟢 极度悲观（逆向买入信号加强）
   ├── "目标价XX元""加杠杆冲" → 🔴 狂热（顶部信号）
   └── 讨论理性、分歧明显 → ⚪ 健康

3. 龙虎榜散户动向
   最近龙虎榜中：
   ├── 散户席位净卖出 + 机构净买入 → 🟢（散户恐慌出，机构接）
   └── 散户席位净买入 + 机构净卖出 → 🔴（散户接盘，机构出货）

4. 小单资金流向
   ├── 小单持续净买入 → 散户在买
   └── 小单持续净卖出 → 散户在卖

   ⚠️ 散户行为是反向指标：
   散户恐慌卖出 = 🟢 好信号
   散户狂热买入 = 🔴 坏信号

散户情绪得分（满分25）：
   ├── 散户极度恐慌（骂声一片+小单净卖出）→ 25分（逆向看多）
   ├── 散户正常讨论 → 15分（中性）
   └── 散户狂热（目标价调高+小单净买入）→ 0分（逆向看空）
```

### 维度 4：综合情绪温度计（5分钟）

#### 4.1 三维加权计算

```
情绪温度计综合计算：

情绪温度 = 市场情绪得分×30% + 个股舆情得分×40% + 散户逆向得分×30%

散户得分使用逆向逻辑：
  散户恐慌 → 散户逆向得分高（逆向看多）
  散户狂热 → 散户逆向得分低（逆向看空）
```

#### 4.2 温度区间与策略

```
┌───────────┬──────────┬────────────────────────────────────┐
│ 温度区间   │ 标签      │ 策略建议                            │
├───────────┼──────────┼────────────────────────────────────┤
│ 0-20      │ 🥶 冰点   │ ⭐ 左侧分批建仓                      │
│           │          │ 特征：散户恐慌卖出、媒体悲观、        │
│           │          │ 无人关注、成交低迷                    │
│           │          │ 策略：每次下跌加仓10%，共3-4次         │
├───────────┼──────────┼────────────────────────────────────┤
│ 21-40     │ 🟢 偏冷   │ ✅ 可以布局                          │
│           │          │ 特征：情绪不高但已稳定、              │
│           │          │ 成交温和、分歧存在                    │
│           │          │ 策略：寻找基本面支撑的买点            │
├───────────┼──────────┼────────────────────────────────────┤
│ 41-60     │ ⚪ 中性   │ ⚪ 维持仓位，不做情绪博弈              │
│           │          │ 特征：多空均衡、消息面平静            │
│           │          │ 策略：正常持有，关注其他维度信号      │
├───────────┼──────────┼────────────────────────────────────┤
│ 61-80     │ 🟡 偏热   │ ⚠️ 右侧跟随，设紧止损                 │
│           │          │ 特征：舆论偏向乐观、散户开始关注、    │
│           │          │ 成交量放大、北向流入                  │
│           │          │ 策略：已有仓位可持有，新买需谨慎       │
├───────────┼──────────┼────────────────────────────────────┤
│ 81-100    │ 🔥 沸腾   │ 🚫 逐步减仓                          │
│           │          │ 特征：全民讨论、散户涌入、            │
│           │          │ 媒体狂热、连续涨停                    │
│           │          │ 策略：每次上涨减仓1/3                 │
└───────────┴──────────┴────────────────────────────────────┘
```

#### 4.3 特殊情形

```
温度陷阱识别：

1. 基本面驱动的"合理升温" vs 情绪驱动的"过热"
   ├── 基本面拐点确认 + 温度上升 → 合理升温，可跟随
   └── 基本面无变化 + 温度飙升 → 情绪炒作，警惕

2. 假冰点（基本面崩溃导致的"该冷"）
   ├── 基本面恶化 + 情绪冰点 → 这是合理的悲观，不是逆向买入信号
   └── 基本面良好 + 情绪冰点 → ⭐ 真正的逆向买入机会

3. 温度与价格的背离
   ├── 价格下跌 + 温度上升 → ⚠️ 散户在抄底，可能还没跌完
   └── 价格上涨 + 温度下降 → 🟢 健康的上涨（情绪未跟上）
```

## 四、模式 B：周度情绪速览（5-10分钟）

```
周度情绪快速扫描：

1. 市场温度快照（3分钟）
   ├── 本周涨停/跌停对比（AKShare stock_zt_pool_em）
   ├── 北向本周净流向（AKShare stock_hsgt_fund_flow_summary_em）
   └── 快速估算市场情绪是否极端

2. 个股舆情快扫（3分钟）
   ├── 本周个股重要新闻（AKShare stock_news_em + WebSearch）
   └── 标记：正面/负面/中性

3. 散户情绪速读（2分钟）
   ├── WebSearch 股吧/雪球最新热门帖
   └── 判断散户情绪倾向

输出：周度温度快报（简短）
  ├── 温度较上周变化：+X / -X
  ├── 是否出现极端信号：是/否
  └── 本周操作情绪建议：左侧/右侧/观望/减仓
```

## 六、核心原则

1. **情绪是反向指标**：极度乐观 = 谨慎，极度悲观 = 机会（但必须配合基本面验证）
2. **区分"合理温度"与"情绪驱动"**：基本面好转带来的升温 ≠ 情绪炒作
3. **温度走势比绝对值重要**：从冰点回暖的信号强度 > 一直中性的信号
4. **散户行为是反向指标，但需要基本面确认**：不能仅凭散户恐慌就逆向买入
5. **情绪面置信度天然偏低**：永远标记 confidence ≤ 70，提醒总分析师情绪面信号有不确定性
