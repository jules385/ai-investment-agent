# /beautify-report — 报告美化

将 Markdown 格式的初次覆盖报告转换为高可读性的 HTML 文件。

## 流程
1. 确认标的代码和名称（如 `002202-金风科技`）
2. 检查以下文件是否存在：
   - `reports/stocks/[代码]-[名称]/01-初次覆盖/初次覆盖报告-*.md`（总分析师报告）
   - `reports/stocks/[代码]-[名称]/01-初次覆盖/子报告-*.md`（四份子报告）
3. 运行 `python tools/md-to-html.py [代码]-[名称]`
4. 告知用户 HTML 文件输出路径

## 输出
- `reports/stocks/[代码]-[名称]/01-初次覆盖/初次覆盖报告-[代码]-完整版.html`
- 深色主题网页，包含封面、目录、投资结论和四份子报告完整附件
