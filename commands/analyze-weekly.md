# /analyze-weekly — 周度跟踪

对指定标的或白名单全部标的进行**周度跟踪**分析。

## 流程

1. **确认标的**：未指定则读取 `reports/白名单.md`，遍历所有核心持仓+卫星持仓
2. **并行启动4个分析师**（轻量模式）：
   - 技术面（周度快速模式：日线+60分钟双框架，≥5 MCP）
   - 筹码面（快速扫描模式：5日主力+北向+融资+龙虎榜，≥3 MCP）
   - 基本面（边际监控模式：行业新闻+个股公告，按颠覆性/重大/轻微/无影响分级）
   - 情绪面（周度速览模式：温度计更新+重大舆情扫描，≥1 MCP）
3. **每个分析师必须对比上次结论**（引用最近深度分析或上周报告的关键判断，标注维持/强化/削弱/逆转）
4. **强制QA**（4并行）
5. **QA通过后，首席合成精简周度报告**：
   - 本周关键变化（≤3句话）
   - 四维信号速览（4行简表）
   - 风险信号
   - 决策：维持/调整？

## 临时深度分析触发

如果任何分析师检测到以下信号，立即停止周度流程，通知用户建议重启深度分析：
- 基本面：🔴颠覆性变化 或 ≥2个关键假设失效
- 筹码面：资金合力方向逆转
- 技术面：价格突破止损位
- 情绪面：温度区间跨越≥2档

## 输出

- 周度跟踪报告 → `reports/stocks/[代码]-[名称]/02-周度跟踪/周度跟踪报告-[日期].md`
- 报告精简格式：只含信号摘要+操作建议，不粘贴完整子报告
