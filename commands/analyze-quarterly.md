# /analyze-quarterly — 季度跟踪

对指定标的进行**季度财务跟踪**。基本面分析师执行三表联动分析（利润表+资产负债表+现金流量表），其余三维做简略辅助。

## 流程

1. 确认标的
2. 并行启动4个分析师：
   - **基本面（三表联动分析）**：Q0读取基准→Q0.5数据获取+交叉验证→Q1利润表增长质量→Q2资产负债表增长代价→Q3现金流利润含金量→Q4三表联动验证→Q5估值更新→Q6逻辑验证→Q7跟踪指标+评级建议
   - 筹码面（简略=周度快速扫描）
   - 技术面（简略=周度快速模式）
   - 情绪面（简略=周度速览模式）
3. 强制QA
4. 首席合成季度报告：
   - 增长质量/增长代价/利润含金量
   - 三表联动验证结果
   - 估值更新 + 升级/降级建议

## 输出

- `reports/stocks/[代码]-[名称]/04-季度跟踪/季度跟踪报告-[日期].md`
