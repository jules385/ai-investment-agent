# /help — A股AI投研系统命令速查

| 命令 | 用途 | 示例 |
|------|------|------|
| `/analyze-initial 代码 名称` | 初次覆盖 | `/analyze-initial 002414 高德红外` |
| `/analyze-weekly` | 周度跟踪 | `/analyze-weekly` |
| `/analyze-monthly 代码` | 月度跟踪 | `/analyze-monthly 002414` |
| `/analyze-quarterly 代码` | 季度三表联动 | `/analyze-quarterly 002414` |
| `/analyze-annual 代码` | 年度战略复盘 | `/analyze-annual 002414` |
| `/analyze-portfolio-weekly` | 组合批量周度跟踪 | `/analyze-portfolio-weekly` |
| `/beautify-report 代码-名称` | 生成HTML报告 | `/beautify-report 002414-高德红外` |
| `/help` | 显示此帮助 | `/help` |

组合管理：`查看组合` / `组合概览` / `哪些标的该跟踪了`
