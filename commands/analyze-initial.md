# /analyze-initial — 初次覆盖分析

请对目标标的进行**初次覆盖**的完整四维深度分析。

## 流程
1. 确认标的代码和名称
2. 并行启动四个子分析师（基本面、筹码面、技术面、情绪面），均为深度模式
3. 等待所有子分析师返回完整报告
4. **🔴 强制QA：并行启动四个数据分析师，逐份检查每份子报告的数据质量**
5. **QA通过后（通过/有条件通过），首席分析师合成：核心投资故事、信号汇总表、矛盾分析与投资建议**
6. 完整粘贴四份子报告（含QA补充数据）

## 输出
- 初次覆盖报告（Markdown），存入 `reports/stocks/[代码]-[名称]/01-初次覆盖/`
- 可选：运行 `/beautify-report` 生成 HTML 版本
